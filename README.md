# Digital-Literacy-Project

Author: Savit Shetty University: VIT Bhopal Course: Digital Literacy
Registration Number – 25BAI11596

This project is part of the Digital Literacy (CSE0001) course, where the objective is to develop essential skills required to effectively and responsibly navigate the digital world. As a Student Digital Ambassador, the project focuses on creating awareness among peers about digital tools, online safety, professional presence, and communication.

The aim of this project is not only to understand theoretical concepts but also to apply them practically using real-world platforms like GitHub, LinkedIn, and Google Workspace. By completing this project, students develop skills in digital communication, online professionalism, cybersecurity awareness, and collaborative tools, which are essential for academic and career growth.

Task 1: Digital Literacy Awareness Infographic
In this task, a one-page infographic is created using tools like Canva to explain the concept of digital literacy and its importance. It highlights key topics such as useful digital tools, safe internet practices, professional online presence, and email etiquette in a visually engaging format.

Task 2: Student Digital Portfolio
This task focuses on building a professional online presence by creating or updating profiles on platforms like GitHub and LinkedIn. It includes setting up a GitHub README and completing essential profile details to establish a strong digital identity for future academic and career opportunities.

Task 3: Coding & Collaboration Platforms
In this task, basic exposure to coding platforms like HackerRank or CodeChef is gained by solving beginner-level problems. Additionally, a Google Form quiz is created to demonstrate collaboration and data collection skills using Google Workspace tools.

Task 4: Email Etiquette & Communication
This task involves drafting professional emails for academic and internship purposes, ensuring proper format, tone, and structure. It also includes creating a checklist of do’s and don’ts for responsible social media usage, promoting effective digital communication.

Task 5: Cybercrime Awareness & Prevention
In this task, a case study on a common cybercrime (such as phishing or UPI fraud) is developed, explaining how it occurs and its impact. A prevention checklist is also created to spread awareness and help students stay safe online.

Overall, this project provides a practical understanding of digital literacy by combining awareness, hands-on experience, and real-world application. Through the completion of all five tasks, it helps in developing essential skills such as creating a professional online presence, communicating effectively, using digital tools efficiently, and staying aware of online risks.

The project also emphasizes responsible and ethical use of technology, which is crucial in today’s digital environment. By working on platforms like GitHub, LinkedIn, and Google Workspace, it prepares students for future academic and professional challenges. Overall, this project acts as a foundation for becoming a confident and responsible digital citizen.
